# Movie Space 📺

Welcome to **Movie Space**, a sleek and powerful React application designed to help you discover and explore TV shows and their cast details with ease. Leveraging modern web technologies like Vite, Tailwind CSS, and Redux, this app ensures a fast and interactive user experience.

## 🌟 Key Features
- 🔎 **Search for TV Shows:** Quickly find shows using TVmaze's API.
- 🎭 **Cast Information:** Explore detailed cast information for each show.
- 🗂️ **Organized Structure:** Modular and scalable architecture using React Router for seamless navigation.
- ⚡ **Fast Performance:** Optimized with Vite for lightning-speed builds and hot module replacement (HMR).
- 🎯 **State Management:** Powered by Redux for efficient data handling.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone <repository-url>
cd movie-space
```

### 2. Install Dependencies
Using **pnpm** for optimized package management:
```bash
pnpm install
```

### 3. Run the Development Server
```bash
pnpm dev
```
Visit `http://localhost:5173` in your browser to explore the app. ✨

### 4. Build for Production
```bash
pnpm build
```

### 5. Preview Your Build
```bash
pnpm preview
```

## 🌊 Styling with Tailwind CSS
- Fully integrated with **PostCSS** and **Autoprefixer** for smooth styling.
- Easily customize your design system via `tailwind.config.js`.

## ⚙️ API Integration
- Utilizes TVmaze API to fetch show details and cast information.
- Efficiently handles asynchronous requests using `axios`.

## 🌍 Navigation & Routing
- React Router provides seamless page transitions:
  - `/` - Show List Page
  - `/:id` - Show Detail Page
